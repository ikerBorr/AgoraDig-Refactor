Ágora Dig

Modern API infrastructure made simple

ÁgoraDig is a modern discussion platform inspired by the ancient Greek Agora, created to support open dialogue and community interaction. Originally developed by CPV05, the project is now undergoing a refactor to transform the monolithic application into a modular and scalable system with a clear separation of layers, improved testability, and greater independence from external frameworks. This modernization enhances maintainability, supports future growth, and provides a better development experience.

Quick Start

Get up and running in minutes

1

Install dependencies

bun i
2

Configure environment

cp .env.example .env.local
3

Start development server

bun run dev #or bun run docker:dev
